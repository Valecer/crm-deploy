## Рекомендации по формату:

- Формат: PNG или SVG (предпочтительно PNG с прозрачным фоном)
- Размер: рекомендуется 200-300px по ширине
- Соотношение сторон: желательно горизонтальное (ширина больше высоты)
- Фон: прозрачный или белый
