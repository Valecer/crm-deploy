# Логотипы клиентов

Эта папка содержит нормализованные логотипы клиентов. Все файлы приведены к коротким ASCII-именам, чтобы их можно было без проблем хранить и передавать (например, в GitHub).

## Правила именования

- `kebab-case`, латиница, без пробелов и спецсимволов;
- максимально коротко, но понятно: `регион-организация-тип.ext`;
- расширение соответствует исходному формату (`.png`, `.jpg`, `.webp`, `.svg` и т.д.);
- новые логотипы добавляйте сразу с коротким именем — переименовывать задним числом не требуется.

При необходимости массового переименования используйте скрипт `rename_files.ps1` (запуск из текущей папки: `powershell -ExecutionPolicy Bypass -File .\rename_files.ps1`). Он содержит словарь «старое → новое имя» и помогает гарантировать единый стиль.

## Рекомендации по подготовке изображений

- Формат: PNG или SVG; допускаются JPG/WebP при отсутствии прозрачного фона.
- Размер: 200–300 px по ширине, без заметной потери качества при масштабировании.
- Фон: по возможности прозрачный либо #FFFFFF.
- Не забывайте оптимизировать изображения (svgo, imagemin) перед добавлением в репозиторий.
